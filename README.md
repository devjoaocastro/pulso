Pulso is an AI agent platform. Agents control your computer, message your team, manage your smart home, and work together in parallel — from any device, any channel, any AI model. Bring your own keys.

## Get Started

```bash
npm install -g @pulso/companion
pulso-companion login
```

Or use the web app at [app.runpulso.com](https://app.runpulso.com).

## Key Features

- **Computer control** — Screenshot, OCR, mouse, keyboard, shell, files, browser. macOS, Windows, Linux.
- **12 channels** — Telegram, WhatsApp, Slack, Discord, Signal, iMessage, Teams, Google Chat, Matrix, LINE, Instagram, Messenger.
- **Multi-agent teams** — Parallel swarms with smart model routing. 140+ templates.
- **Knowledge graph** — Semantic memory with auto-extraction and natural decay.
- **200+ AI models** — Anthropic, OpenAI, Google, DeepSeek, Mistral, Ollama, OpenRouter. BYOK.
- **Smart home** — Philips Hue, Sonos, Spotify, Home Assistant.
- **Automations** — Event-driven triggers that run full agent loops 24/7.
