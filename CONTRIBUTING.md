# Contributing to Pulso

Thank you for your interest in Pulso.

## Source Code

Pulso is proprietary software. The source code is not available for public contribution at this time.

## How You Can Help

### Report Bugs

Found a bug? Please [open an issue](https://github.com/usepulso/pulso/issues/new?template=bug_report.md) with:

- Steps to reproduce
- Expected behavior
- Actual behavior
- Platform and browser/app version

### Request Features

Have an idea? [Open a feature request](https://github.com/usepulso/pulso/issues/new?template=feature_request.md) with:

- Description of the feature
- Use case and motivation
- Any examples from other products

### Security Issues

Please see [SECURITY.md](SECURITY.md) for reporting security vulnerabilities. **Do not open public issues for security reports.**

### Feedback

General feedback and questions: support@runpulso.com
